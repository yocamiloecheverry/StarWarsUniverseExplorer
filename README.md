# Star Wars Universe Explorer

App Android (Kotlin + Jetpack Compose) que consume la API SWAPI para explorar:
- Personajes
- Películas
- Planetas
- Naves
- Vehículos

## Tecnologías
- Kotlin
- Jetpack Compose
- MVVM
- Retrofit
- Material 3

## Funcionalidades
- Listas + búsqueda (personajes)
- Detalles (personajes, películas, planetas, naves, vehículos)
- Estados: Loading / Success / Error
